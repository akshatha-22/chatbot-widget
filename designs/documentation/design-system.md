# Design System

## Overview

The Floating Chatbot Launcher Design System provides a comprehensive set of guidelines and components for building consistent user interfaces.

## Color Palette

### Primary Colors
- **Primary Blue**: `#0ea5e9` - Main action color
- **Primary Dark**: `#082f49` - Dark backgrounds and text

### Semantic Colors
- **Success**: `#10b981` - Success states and confirmations
- **Error**: `#ef4444` - Error states and alerts
- **Warning**: `#f59e0b` - Warning messages and cautions

### Neutral Colors
- **White**: `#ffffff` - Backgrounds
- **Gray 50**: `#fafafa` - Light backgrounds
- **Gray 900**: `#171717` - Text on light backgrounds

## Typography

### Font Family
- **Sans-serif**: System fonts for optimal readability
- **Monospace**: For code and technical content

### Font Sizes
- **Body**: 16px (base)
- **Small**: 14px
- **Large**: 18px+

### Font Weights
- **Normal**: 400
- **Medium**: 500
- **Semibold**: 600
- **Bold**: 700

## Spacing

Consistent 4px base unit:
- **xs**: 4px
- **sm**: 8px
- **md**: 12px
- **lg**: 16px
- **xl**: 24px
